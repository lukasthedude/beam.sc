# beam.sc

The Beam website — a single static page, served by GitHub Pages at **https://beam.sc**.

```
index.html   the page
styles.css   Inter, Apple-minimal, light + dark (follows the device)
assets/      self-hosted Inter font, favicon (the "B"), link-preview image ("Beam")
CNAME        custom domain (beam.sc)
.nojekyll    serve files as-is (no Jekyll processing)
```

No build step. Edit, commit, push — GitHub Pages redeploys automatically.
